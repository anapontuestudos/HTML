HTML = Hyper Text Markup Languagem

Linguagem de marcação de hipertexto


  • Essa linguagem, ao contrário do Javascript ou PHP, não é uma linguagem de programação mas sim uma linguagem de marcação pois ela trabalha com tags;
  • Essa linguagem é responsável por organizar o conteúdo das páginas web;
  • Hipertexto se refere aos links que conectam as páginas.
